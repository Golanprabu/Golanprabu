About Me

Pertama kali saya mulai mempelajari coding ketika saya menonton sebuah video di TikTok dari Dea Alfarizil. Setelah itu, saya mulai mencoba-coba di perangkat seluler saya. Rasa tertantang kemudian mendorong saya untuk mendalami ilmu ini lebih lanjut, dan akhirnya, saya memutuskan untuk kuliah di sebuah universitas yang menawarkan program studi dalam bidang coding/program.


Web yang Anda lihat sekarang adalah hasil dari perjalanan pembelajaran saya. Saya mendapatkan bimbingan melalui video belajar web programming di Universitas Pasundan (Unpas).

Saya adalah seorang pelajar di SMK Negeri 1 Obaa dan telah menyelesaikan pendidikan saya di sana. Selain coding, saya juga memiliki hobi melukis menggunakan perangkat Android. Ketertarikan saya terhadap seni dimulai saat saya belajar Corel Draw di kelas 10. Meskipun laptop saya mengalami kerusakan, saya tidak berputus asa dan memanfaatkan perangkat Android saya. Awalnya, saya menggunakan aplikasi bernama Color, namun kemudian beralih ke aplikasi yang lebih mudah digunakan, yaitu Ibis Paint. Sejak saat itu, saya terus menggunakan Ibis Paint untuk melukis. Di bawah ini adalah contoh karya seni yang pernah saya buat.

Saat ini, saya tengah menjalani studi di Universitas Dipa Makassar dan berencana menyelesaikan masa kuliah saya dalam waktu empat tahun. Namun, saya berharap bisa menyelesaikan program studi tepat pada waktu atau bahkan lebih cepat. Selama di perguruan tinggi, saya juga sedang fokus mempelajari bahasa pemrograman Python.


Selama perjalanan pembelajaran saya, saya telah menciptakan beberapa program menggunakan Python, antara lain adalah Python AI Assistant dan DRIODcam Python. Anda dapat melihat kode-kode dari proyek-proyek tersebut pada halaman projects web yang tersedia.

Saya sangat bersemangat untuk terus meningkatkan kemampuan dan pengetahuan saya dalam dunia coding dan programming serta senang berbagi karya-karya saya dengan orang lain.
